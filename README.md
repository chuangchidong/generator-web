# 介绍
  代码生成器的web界面，完成数据库表自动生成代码</br>

## 浏览器支持
* IE 9+
* Firefox（最新）
* Chrome（最新）
* Safari（最新）
## 运行项目
  进入generator-web 运行一下命令安装依赖包。
```
npm install
```
  安装完成之后，开始运行项目。
```
npm run dev 
```
## 编译项目
  运行一下命令：
```
npm run build
```
  自动会在浏览器中打开 http://localhost:8080/#/login 的地址：</br>
  用户名： 任意输入（未做校验）</br>
  密码： 大于6个字符。
